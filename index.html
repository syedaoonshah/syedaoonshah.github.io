RGB-DINO-Gaze: Label-Efficient Gaze Estimation via Self-Supervised Learning

Master's Thesis

Supervisor: Prof. Xinbo Zhao, Northwestern Polytechnical University

Master's Thesis, Northwestern Polytechnical University, 2025

A novel three-phase semi-supervised framework for gaze estimation that significantly reduces the need for labeled training data while maintaining high accuracy.

Key Contributions:

  • Designed novel three-phase semi-supervised framework combining RGB-MEM (Masked Eye Modeling) pretraining, dual self-supervision with consistency regularization, and MAML-based meta-adaptation
  • Achieved 1.70° mean angular error using only 50% labeled data, outperforming fully supervised baseline (1.79° with 100% labels) by 5.3%
  • Developed eye-region-aware masking strategy preserving anatomically critical pupil and eyelid regions during unsupervised reconstruction
  • Implemented DINOv2-based architecture with frozen early layers and lightweight bottleneck adapter (768→64→768) for efficient few-shot user calibration
